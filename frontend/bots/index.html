VK community setup (if the bot doesn't reply)
  1. Откройте сообщество → Управление.
  2. Сообщения → включите «Сообщения сообщества» (открытые или по кнопке).
  3. Работа с API → включите Long Poll API (Callback API можно не использовать).
  4. В приложениях VK создайте приложение, привяжите сообщество, выдайте ключ доступа сообщества с правами: управление сообществом, сообщения сообщества.
  5. Токен вставьте при создании бота (формат vk1.a...).
  6. На сайте: «Запустить» → в VK напишите боту привет или меню (команда /start не обязательна).
